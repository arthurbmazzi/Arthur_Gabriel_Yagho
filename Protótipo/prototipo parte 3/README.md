# Jogo POO

Helper para o MelonJS


### 📋 Pré-requisitos

Node + Npm e Live server ou Http server


### 🔧 Instalação

Download do MelonJs de https://github.com/melonjs/melonJS/releases seguido de:
```
$ npm install
```

ou rodar o comando:
```
$ npm install melonjs
```

## ⚙️ Executar

```
$ npm run build
```
